In order to run the demo jupyter notebook file 'notebook_intro.ipynb'
you need to do the follwoin (assuming you have installed a jupyter
notebook - tensorflow environment):


1. Start a shell (linux/macos) or a command window (windos)
and activate you tensorflow environment

2. Start jupyter notebook

e.g.

..> jupyter notebook

3. Find the folder where you save the two files
'notebook_intro.ipynb'
'auto.png'

4. Click on the file 'notebook_intro.ipynb'
